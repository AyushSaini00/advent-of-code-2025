## Running Solutions
First, you'd need to cd into a day directory. example: `cd day-1`

### TypeScript

```bash
cd ts
yarn install
yarn dev 

# to run the tests
yarn test
```

### Go

```bash
cd go
go build && ./day-1

# to run the tests
go test ./...
```

### Python

```bash
cd python
python3 main.py
```
